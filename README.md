# Quiz Game (Python Tkinter)

## 📌 Project Description

Quiz Game is a simple desktop application built using *Python and Tkinter*. The application presents multiple-choice questions to the user and calculates the final score based on correct answers. It provides an interactive graphical user interface where users can start the quiz, select answers, and view their results at the end.

## 🚀 Features

* Interactive GUI built with *Tkinter*
* Multiple-choice quiz questions
* Automatic score calculation
* Start, restart, and exit options
* Simple and user-friendly interface

## 🛠️ Technologies Used

* Python
* Tkinter (GUI library)

## ▶️ How to Run the Project

1. Install Python (if not installed).
2. Download or clone the repository.
3. Run the Python file.

bash
python Quiz_Game_Project.py


The quiz window will open where you can start the game.

## 🎮 How It Works

1. Click *Start Quiz*.
2. Read the question and select the correct option.
3. Click *Submit* to move to the next question.
4. At the end, the application shows your *final score*.
5. You can restart the quiz or exit the application.

## 📊 Example Questions
